
-- SUMMARY --

* Nodetype trigger provides additional triggers to associate actions with. Creation and deletion of nodes can launch triggers based on the node type.


-- REQUIREMENTS --

* The core trigger module.


-- INSTALLATION --

* Install the module as usual.
* Set the node types for which you want to make the triggers available at admin/settings/nodetype-trigger.
* Associate actions with custom triggers at admin/build/trigger/nodetype_trigger.
* If the Nodetype trigger tab is not present at admin/build/trigger/nodetype_trigger clear cache and reload the page.
